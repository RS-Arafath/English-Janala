# English Janala

A modern web app to **learn English words with Bangla meaning** and **correct English pronunciation**. Perfect for beginners who want to improve vocabulary and speaking skills easily.

---

## 🚀 Features

- ✅ Learn English words with Bangla translation  
- ✅ Listen to correct English pronunciation  
- ✅ Simple, responsive, and user-friendly interface  
- ✅ Built with **HTML**, **Tailwind CSS**, **DaisyUI**, and **JavaScript**

---

## 💻 Tech Stack

| Frontend | Styling | Programming | Deployment |
|----------|---------|------------|------------|
| HTML     | Tailwind CSS, DaisyUI | JavaScript | Vercel |

---

## 🎯 Usage

1. Open the app: [English Janala](https://english-janala-eta.vercel.app/)  
2. Browse English words  
3. See Bangla meaning and listen to pronunciation
